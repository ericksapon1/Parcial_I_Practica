En esta práctica se hizo un programa en C++ en donde se define una estructura básica con un dato entero y un puntero para instanciar dinámicamente tres nodos en tiempo de ejecución utilizando el operador new.
Tras solicitar y almacenar los valores ingresados por el usuario en cada espacio de memoria, se procede a enlazar los nodos secuencialmente para formar una lista enlazada simple que termina en un valor nulo. 
Se uso un puntero auxiliar para recorrer la estructura e imprimir tanto mi nombre como los datos guardados, se finaliza el proceso con la liberación adecuada de los recursos mediante el operador delete para garantizar una correcta gestión de la memoria del sistema.
